# ADR 0001: Webアプリの既定デザイン方向

## 状態

採用

## 文脈

この開発キットは、Webアプリを短い反復で作る際の共通デザイン方向を持たなかった。テーマや画面遷移の判断が依頼ごとに揺れると、実装前の合意と利用者体験が不安定になる。

## 決定

Webアプリの明示的なブランド指定がない場合、現行 NTT DATA の公式サイトおよび現行CIを参照した青系テーマを既定にする。白または明るいニュートラルを基調にし、青は主操作、状態、重要情報の識別に限定する。

既存ブランド、デザインシステム、または別テーマが明示された場合は、その指定を既定テーマより優先する。ロゴ、商標アセット、固定の公式カラー値はキットに同梱しない。現行CIは変わり得るため、具体的な実装時に公式情報を参照する。

## 結果

Webアプリには一貫した信頼感のあるデフォルトがある一方、利用者のブランド要件を妨げない。ブランドの正確な資産管理は、この開発キットの責務に含めない。
