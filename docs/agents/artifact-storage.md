# ローカル成果物の保管

ハーネスが生成する一時ファイルは、機能ごとに `.scratch/<機能名>/` へ保存する。このディレクトリには、ローカルの仕様・チケット、テスト実行結果、スクリーンショット、ログ、調査メモ、試作物を置ける。

`.scratch/` は Git 管理しない。成果物を共有・恒久化する必要が生じた場合は、内容を整理して `docs/`、ソースコード、または GitHub Issue に明示的に移す。

再現に必要なテストコードは Git 管理する。一方、`coverage/`、`test-results/`、`playwright-report/`、`node_modules/`、`.data/` のような再生成可能な出力は Git 管理しない。
